# Novia Clone - System Dashboard

This is a React + Vite project reproducing the Novia system dashboard.

## Features
- **Responsive Layout**: Sidebar, Header, and Grid-based widgets.
- **Visual Effects**: Scanlines, glitch text, typing effects, heartbeat monitor.
- **Components**:
  - `TacticalAsset`: Displays character details with image overlay effects.
  - `SystemStatus`: Real-time canvas-based heartbeat loop.
  - `Sidebar`: Navigation with hover states.
  - `SystemLogs`: Auto-scrolling simulated terminal logs.

## Setup

1. **Install Dependencies** (if not done already):
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Open in Browser**:
   Visit `http://localhost:5173`
